# 答案之书 · The Book of Answers

> 给正在人生低谷、遭遇挫折与迷茫的人，一句能带来希望与力量的话。

翻一页，宇宙会从文学、电影、音乐与诗歌里，为你挑出一句此刻最需要的光。

🌐 **在线体验：** https://carashen-data.github.io/answer-book/

---

## ✨ 这是什么

一个**纯前端、零依赖、离线可用**的单文件网页应用。打开即用，双击 `index.html` 即可在浏览器运行，无需服务器、无需联网、无需安装。

它想做的，是在你最丧的时候，做一个安静递话的人——不鸡汤、不说教，只是把某个人在黑暗里写下的光，翻到你面前。

---

## 🌿 功能

- **🎲 先被看见**：翻书前先选今天的感觉——挫败 / 失落 / emo / 失恋 / 工作不顺 / 迷茫 / 自我否定 / 压力爆棚 / 无力 / 无聊，或「随便翻翻」。
- **📖 翻开答案**：3D 翻书动画后，随机浮现一句语录（中文 + 英文 + 出处），且会**优先匹配你选的情绪**，更懂你。
- **🎞️ 治愈小动画**：每种情绪配一段内联 SVG 小动画与一句温柔旁白（苗破土、雨转星、日出拨云……全部离线）。
- **🌿 心境处方**：翻开后，按心情推荐「一本书 + 一部电影 + 一首歌 + 一个活动」，各附一句为什么适合此刻。
- **💛 收进心里**：本地收藏夹（`localStorage`），可查看、复制、导出 .txt、清空；关掉网页也还在。
- **📋 抄下这句**：可手动选中复制，或一键复制 / 下载为 .txt。
- **🖼️ 分享卡片**：把「一句话 + 处方」用 Canvas 生成竖版海报图（3:4），一键保存，方便发朋友圈 / 社群。
- **↩ 回到首页**：随时合上书，重新选择心情再翻。

## 🚀 用法

### 本地使用
直接双击 `index.html`，或在浏览器打开即可。

### 部署到 GitHub Pages
1. 把本仓库 Fork 或克隆到你的 GitHub。
2. 在仓库 **Settings → Pages** 中，Source 选择 `main` 分支、目录 `/ (root)`。
3. 稍等片刻，即可通过 https://carashen-data.github.io/answer-book/ 访问。

### 部署到其他静态托管
因为是纯静态单文件，直接把 `index.html` 丢到任意静态托管（Vercel / Netlify / CloudStudio / Nginx）即可。

---

## 🛠️ 技术说明

- 单个 `index.html`，内联 CSS + 原生 JavaScript，**无任何外部依赖、无 CDN、无追踪**。
- 所有语录、情绪处方、动画均为内置数据，完全离线。
- 收藏功能使用浏览器 `localStorage`。
- 分享卡片使用 Canvas 2D 绘制，中英混排按字数 / 单词自动换行。
